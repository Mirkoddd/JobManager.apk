배터리 수명을 연장하려면

배터리 수명은 CPU 사용률과 관련이 있습니다. 사용하지 않는 애플리케이션은 모두 종료하세요. 일부 애플리케이션은 사용 중인 프로그램 목록에 표시되지 않을 수 있습니다

1. 백그라운드에서 실행 중인 일부 애플리케이션은 불필요한 배터리 소모의 원인이 됩니다. 애플리케이션을 종료하려면 홈 버튼 대신 취소 버튼을 사용하세요. (취소 버튼은 대부분의 애플리케이션을 종료하지만 홈 버튼은 애플리케이션을 대기 모드로 전환합니다.)

2. 애플리케이션 모니터 위젯으로 현재 실행 중인 애플리케이션의 수를 확인할 수 있습니다. (일부 애플리케이션은 표시되지 않을 수 있습니다.) 알 수 없거나 보안되지 않은 애플리케이션을 설치하면 과도한 배터리 소모의 원인이 될 수 있으므로 주의하세요

3. 구글 계정을 설정하면 데이터 통신은 백그라운드에서 실행됩니다. 대기 시간을 연장하려면 설정 > 계정 및 동기화에서 자동 동기화를 해제하고 필요할 때 수동으로 동기화하세요

4. 장시간 켜 놓은 Wi-Fi와 블루투스가 종종 배터리 소모의 주 원인이 됩니다

5. 배터리가 지나치게 빨리 소모되거나 제품이 비정상적으로 작동할 경우, 리셋을 위해 전원키를 8초이상 누르거나 전원종료를 이용하여 전원을 껐다 다시 켜세요

RAM 관리자 정보

안드로이드 OS 애플리케이션은 빠른 재실행을 위하여 종료된 후에도 메모리의 일부를 사용할 수도 있습니다. 메모리가 부족해지면 LMK(Low Memory Killer)가 LRU(Least recently used) 목록을 이용하여 자동으로 메모리를 비웁니다. 메모리를 수동으로 비우면 애플리케이션을 시작하는 데 더 많은 시간이 걸립니다

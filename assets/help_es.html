Cómo prolongar la duración de la batería

La duración de la batería está relacionada con el uso de CPU. Cierre todas las aplicaciones innecesarias. Es posible que algunas aplicaciones no aparezcan en la lista de programas en ejecución

1. Algunas aplicaciones en segundo plano pueden provocar un consumo de batería no deseado. Para cerrar una aplicación, utilice la tecla de retroceso en lugar de la tecla de inicio. (La tecla de retroceso cerrará la mayor parte de las aplicaciones, mientras que la de inicio pondrá las aplicaciones en modo de espera.)

2. Puede ver el número de aplicaciones que están ahora en ejecución con el asistente de control de programas. (Es posible que algunas aplicaciones no aparezcan.) Tenga cuidado al instalar aplicaciones no seguras o desconocidas, pues puede provocar un consumo excesivo de batería

3. Después de configurar una cuenta de Google, la comunicación de datos se lleva a cabo en segundo plano. Para ampliar el tiempo de espera, desactive la sincronización automática en Ajustes > Cuentas y sincronización, y efectúe la sincronización manual cuando sea necesario

4. Mantener las funciones de Wi-Fi y Bluetooth activadas es con frecuencia la principal causa de fugas en la batería

5. En caso de que se produzca un mal funcionamiento del teléfono o un consumo excesivo de batería, reinicie el teléfono. Para ello, pulse a la vez la tecla de encendido y la tecla de subir volumen durante 8 segundos o mantenga pulsada la tecla de encendido y seleccione Apagar

Acerca del administrador de RAM

Para habilitar el reinicio rápido, las aplicaciones con el SO Android permanecen en la memoria, aunque estén cerradas. Si no hay suficiente memoria, el LMK (Low Memory Killer) libera la memoria automáticamente con la lista de LRU (menos usados recientemente). Si libera la memoria de forma manual, puede que las aplicaciones necesiten más tiempo para iniciarse.
